## 一、数据来源与处理

### 1.1 WHO FluNet 哨点监测数据

| 项目 | 说明 |
|------|------|
| **数据源** | WHO Global Influenza Surveillance and Response System (GISRS)，中国（country_code=`CHN`） |
| **时间范围** | 2015–2024 年（可在 `config.yaml` 中配置 `year_from` / `year_to`） |
| **指标** | H3N2 亚型阳性率 = 阳性标本数 / 检测总数（周粒度） |
| **下载脚本** | `data/fetch_flunet.py` |
| **处理后数据** | `data/processed/weekly_positivity.csv` |

> **注意**：模型输出 `I_rate` 是校园内活跃感染人数占总人口的比例，而 FluNet 阳性率是就诊阳性率，两者量纲不同。因此校准后通过观测比例因子 ρ 将两者对齐（详见第五节）。

### 1.2 数据预处理

`prepare_obs_timeseries` 函数完成以下转换：

1. **时间对齐**：将真实日历时间映射为模拟内相对天数
   - 以一年中第 `t_start_doy` 天（默认 305 = 11月1日）作为模拟起点 $t=0$
   - 每条 FluNet 周数据计算 `t_sim = (dayofyear - t_start_doy) % 365`

2. **多年均值聚合**（`aggregate=True`）：
   - 将 522 周原始数据按 7 天 bin 取均值，压缩至约 27 个周次均值点
   - 消除年际噪声，使校准 R² 更稳定

3. **MAPE 过滤**：验证时仅在 `h3n2_pos_rate > 1%` 的周次计算 MAPE，避免低阳性率周的分母过小问题

## 二、人口结构与舱室模型

### 2.1 两群体设定

```
总人口 N = N1 + N2 = 27,000 + 3,000 = 30,000 人
├── 学生群体（群体1）：N1 = 27,000
└── 教职工群体（群体2）：N2 = 3,000
```

### 2.2 SEIQR 舱室定义

每个群体有 5 个舱室，合计 **10 维状态向量**：

```
[S1, E1, I1, Q1, R1,  S2, E2, I2, Q2, R2]
 ├──群体1（学生）─┘       └──群体2（教职工）─┘
```

| 舱室 | 含义 | 状态特点 |
|------|------|----------|
| **S** (Susceptible) | 易感者 | 未感染、无免疫，可被感染 |
| **E** (Exposed) | 潜伏期暴露者 | 已感染但尚无传染性 |
| **I** (Infectious) | 感染者 | 有传染性，可传播疾病 |
| **Q** (Quarantined) | 隔离者 | 被发现并隔离，丧失传染性 |
| **R** (Recovered) | 康复者 | 已康复，获得免疫力（单季 ω=0 不返回 S）|

### 2.3 初始条件

```python
疫苗接种者比例 = vax_coverage × vax_efficacy ≈ 0.32% × 27% ≈ 0.09%
S0_1 = N1 × (1 - 疫苗等效免疫比例) - I0_1
R0_1 = N1 × 疫苗等效免疫比例        # 疫苗保护直接进入 R 舱
```

## 三、核心方程组

### 3.1 传播力（力感染，Force of Infection）

$$\lambda_i(t) = \beta(t) \cdot c(t) \cdot \sum_{j} C_{ij} \cdot \frac{I_j}{N_j}$$

其中：
- $\beta(t)$：季节传播系数（随时间变化的"基础传染率"）
- $c(t)$：学期调制系数（假期降低接触）
- $C_{ij}$：2×2 接触矩阵，日均有效接触次数
- $I_j/N_j$：群体 j 的感染率（作为感染压力）

### 3.2 两群体 SEIQR 微分方程

**学生群体（$i=1$）：**
```
dS₁/dt = -λ₁(t) × S₁ + ω × R₁
dE₁/dt =  λ₁(t) × S₁ - σ × E₁
dI₁/dt =  σ × E₁ - (γ × (1-p_iso) + α × p_iso) × I₁
dQ₁/dt =  α × p_iso × I₁ - γ_Q × Q₁
dR₁/dt =  γ × (1-p_iso) × I₁ + γ_Q × Q₁ - ω × R₁
```

**教职工群体（$i=2$）：**（方程形式同上，下标替换为2）

**参数物理意义：**

| 参数 | 含义 | 典型值 |
|------|------|--------|
| $\beta_0$ | 每次接触的传播概率（待拟合） | 0.015 |
| $\sigma$ | 潜伏期转感染率，$1/\sigma$ = 潜伏期天数 | 0.50（2天） |
| $\gamma$ | 感染者自然康复率，$1/\gamma$ = 感染期天数 | 0.25（4天） |
| $\gamma_Q$ | 隔离者康复率 | 0.20（5天） |
| $\alpha$ | 病例隔离率（被发现的感染者比例） | 0.15 |
| $p_{iso}$ | 被发现者成功隔离的比例 | 0.40 |
| $\omega$ | 免疫衰退率（单季=0） | 0.0 |

### 3.3 接触矩阵

$$C = \begin{pmatrix} c_{11} & c_{12} \\ c_{21} & c_{22} \end{pmatrix} = \begin{pmatrix} 18.0 & 2.0 \\ 2.0 & 8.0 \end{pmatrix}$$

| 元素 | 含义 |
|------|------|
| $c_{11}=18$ | 学生-学生：课堂+宿舍+食堂+课外 ≈ 18次/天 |
| $c_{12}=2$ | 学生-教职工：课堂+办公室 ≈ 2次/天 |
| $c_{21}=2$ | 教职工-学生（对称） |
| $c_{22}=8$ | 教职工-教职工：办公室+会议 ≈ 8次/天 |

## 四、季节强迫与时间调制

### 4.1 β(t) 双谐波季节函数

$$\beta(t) = \beta_0 \times \left[1 + \delta_1 \cos\left(\frac{2\pi t_{abs}}{365} + \phi_1\right) + \delta_2 \cos\left(\frac{4\pi t_{abs}}{365} + \phi_2\right)\right]$$

- $t_{abs} = (t_{start\_doy} + t) \mod 365$：一年中的绝对天数
- $\delta_1, \phi_1$：年周期振幅和相位（主要流行高峰在冬季）
- $\delta_2, \phi_2$：半年周期振幅和相位（捕捉秋冬+春末双峰）
- 共 4 个季节参数通过 FluNet 数据拟合得到

### 4.2 学期调制 c(t)

假期期间接触大幅减少（学生离校）：

| 假期 | 时间段（day-of-year） | 调制系数 |
|------|----------------------|---------|
| 寒假 | 15–42（1月15日–2月11日）| 0.30 |
| 暑假 | 197–243（7月16日–8月31日）| 0.30 |
| 国庆 | 274–281（10月1日–8日）| 0.65 |
| 五一 | 121–125（5月1日–5日）| 0.75 |
| 学期中 | 其他时间 | 1.00 |

## 五、方程求解

### 5.1 数值方法

使用 `scipy.integrate.odeint`（LSODA 算法，自动检测刚性）：

```python
sol = odeint(seiqr_rhs, y0, t_eval, args=(p,), rtol=1e-6, atol=1e-8)
```

- **刚性检测**：LSODA 自动在 GFESIRK（非刚性）和 ADAMS（前馈）方法间切换
- **求解区间**：$t \in [0, t_{days}]$，默认 180 天（覆盖一个冬春流行季）
- **非负保护**：求解后 `sol = np.maximum(sol, 0.0)` 防止数值下溢

### 5.2 派生指标

从状态轨迹计算：

| 指标 | 计算公式 |
|------|----------|
| 总感染人数 | $I_{total} = I_1 + Q_1 + I_2 + Q_2$ |
| 活跃感染率 | $I_{rate} = I_{total} / N$ |
| 累计发病率 | $AR = (R_1 + R_2 - R_{初始}) / (N - R_{初始})$ |
| 峰值时间 | $\arg\max_t I_{total}(t)$ |
| 翻倍时间 | $\ln(2) / r$（指数增长期线性拟合）|

## 六、参数拟合（校准）

### 6.1 待拟合参数

| 参数 | 下界 | 上界 | 含义 |
|------|------|------|------|
| $\beta_0$ | 0.003 | 0.08 | 基础传播系数 |
| $\delta_1$ | 0.00 | 0.70 | 年季节振幅 |
| $\delta_2$ | 0.00 | 0.80 | 半年季节振幅 |
| $\phi_1$ | $-\pi$ | $\pi$ | 年相位（弧度）|
| $\phi_2$ | $-\pi$ | $\pi$ | 半年相位（弧度）|
| $\alpha$ | 0.01 | 0.80 | 病例隔离率 |

固定参数（来自文献）：$\sigma=0.50$, $\gamma=0.25$, $\gamma_Q=0.20$, $p_{iso}=0.40$

### 6.2 目标函数

$$\min_{\theta} \sum_i w_i \left[ I_{rate}(t_i; \theta) - \text{obs}_i \right]^2$$

其中 $w_i = \sqrt{\text{obs}_i + 0.01}$ 为权重（高阳性率周权重更大）。

### 6.3 求解器

使用 `lmfit.minimize`（方法：`leastsq`），残差函数：

```python
def _residuals_func(lm_params, obs_t, obs_y, p_base, t_sim):
    p = p_base.update(**{name: lm_params[name].value for name in lm_params})
    df_sim = solve_seiqr(p, t_eval=t_sim)
    sim_at_obs = interp1d(df_sim["t"], df_sim["I_rate"])(obs_t)
    return np.sqrt(obs_y + 0.01) * (sim_at_obs - obs_y)
```

### 6.4 观测比例因子 ρ

模型输出 $I_{rate}$（活跃感染率）与 FluNet 阳性率量纲不同。拟合结束后计算最优缩放因子：

$$\rho^* = \frac{\sum I_{rate}(t_i) \cdot \text{obs}_i}{\sum I_{rate}(t_i)^2}$$

该 $\rho$ 在验证阶段作为 `scale` 参数传入，使模型曲线与观测数据量纲对齐。

### 6.5 收敛判断

`lmfit.success=False`（"Tolerance too small"）仅表示无法估计误差棒，**最优参数仍然有效**，代码始终使用最佳参数进行后续计算和验证。

## 七、模型验证指标

`compute_metrics` 计算以下指标：

| 指标 | 公式 | 阈值意义 |
|------|------|----------|
| **RMSE** | $\sqrt{\frac{1}{n}\sum(\text{sim}-\text{obs})^2}$ | 越小越好 |
| **MAPE** | $\frac{100\%}{n}\sum|\frac{\text{sim}-\text{obs}}{\text{obs}}|$ | 流行周 MAPE < 60% 为可接受 |
| **R²** | $1 - \frac{\sum\text{res}^2}{\sum(\text{obs}-\bar{\text{obs}})^2}$ | > 0 表示优于均值基准；> 0.4 为良好 |
| **Pearson r** | $\corr(\text{sim}, \text{obs})$ | 趋势一致性 |

## 八、敏感性分析（PRCC）

### 8.1 方法

**偏秩相关系数（PRCC）**：在 Latin Hypercube 采样（LHS）基础上，控制其他参数后计算单一参数与输出的单调相关性。

$$Y = f(X_1, X_2, \ldots, X_k)$$

对 $X_j$ 排秩后，用其余变量回归的残差与 $Y$ 排秩的相关性作为 PRCC 值，取值 $[-1, 1]$。

### 8.2 分析参数（10个）

`beta0, sigma, gamma, alpha, p_iso, delta1, delta2, vax_coverage, c11, c12`

### 8.3 输出指标

| 输出 | 含义 |
|------|------|
| `peak_infection_rate` | 峰值感染率 $I_{peak}/N$ |
| `total_attack_rate` | 累计发病率 $AR$ |
| `peak_timing_days` | 峰值出现时间（天）|

### 8.4 采样配置

```yaml
sensitivity:
  n_samples: 1000       # LHS 采样数
  param_variation: 0.30 # 参数基础值 ±30%
```

## 九、干预措施建模

### 9.1 干预措施参数

| 措施 | 参数 | 效果 |
|------|------|------|
| 口罩 | `mask_level` ∈ [0,1] | $\beta \leftarrow \beta \times (1 - 0.17 \times \text{mask\_level})$ |
| 通风 | `ventilation` ∈ [0,1] | $\beta \leftarrow \beta \times (1 - 0.40 \times \text{ventilation} \times \text{indoor\_weight})$ |
| 消毒 | `disinfection` ∈ [0,1] | $\beta \leftarrow \beta \times (1 - 0.05 \times \text{disinfection})$ |
| 疫苗 | `vaccination` ∈ [0,1] | 等效覆盖率 $= 0.32\% + (\text{target} - 0.32\%) \times \text{vaccination}$ |
| 隔离加强 | `isolation_rate` ∈ [0,1] | $\alpha \leftarrow \alpha \times (1 + 2 \times \text{isolation\_rate})$ |
| 在线教学 | `online_teaching` ∈ [0,1] | $c_{11} \leftarrow c_{11} \times (1 - 0.75 \times \text{online\_teaching})$ |
| 聚集限制 | `activity_limit` ∈ [0,1] | 场所权重重分配，限制高密度活动 |

### 9.2 场所接触权重

| 场所 | 权重 | 说明 |
|------|------|------|
| 宿舍 | 0.30 | 密切接触 |
| 教室 | 0.25 | 集体暴露 |
| 食堂 | 0.10 | 短暂接触 |
| 户外 | 0.35 | 低密度，传播贡献小 |

## 十、多目标优化

### 10.1 目标函数

$$F = w_1 \cdot \frac{AR}{AR_{base}} + w_2 \cdot \frac{PIP}{PIP_{base}} + w_3 \cdot \text{Cost}$$

| 符号 | 含义 | 权重 $w_i$ |
|------|------|------------|
| $AR$ | 累计发病率 | 0.50 |
| $PIP$ | 峰值感染压力（峰值感染率）| 0.30 |
| Cost | 综合成本评分 [0,1] | 0.20 |

### 10.2 优化策略

**网格扫描（Grid Search）**：
- 7 个控制变量，每变量 5 个离散水平（0, 0.25, 0.5, 0.75, 1.0）
- 总组合数：$5^7 = 78,125$ 种
- 过滤条件：$\text{Cost} \leq 0.65$
- 按 $F$ 值升序排列，输出 Top-20 方案

**差分进化（differential_evolution）**：
- 连续空间全局优化，适合精细求解
- 最大迭代 100 次，群体大小 10

### 10.3 成本模型

综合成本 = 7 项干预措施成本之和（每项归一化至 [0,1]），由 `intervention/cost_model.py` 计算。

## 十一、全流程架构（run_all.py）

```
Step 0: 参数初始化
  ├── 从 config.yaml 加载基础参数
  └── 从 seasonal_params.json 更新季节参数

Step 1: 基础求解
  ├── 运行 SEIQR ODE（默认参数）
  ├── 计算 R₀ 和群体免疫阈值（HIT）
  └── 输出：流行曲线图、β(t) 季节图、R₀ 热力图

Step 2: 参数拟合（若 FluNet 数据存在）
  ├── 从 seasonal_params.json 初始化季节参数
  ├── lmfit leastsq 最小二乘拟合
  ├── 计算最优观测比例 ρ
  └── 保存最优参数 → best_params.json

Step 3: Bootstrap 置信区间（500次重采样）
  └── 参数 95% CI + 轨迹置信带

Step 4: 模型验证
  ├── 计算 RMSE / MAPE / R² / Pearson r
  └── 输出：验证图（模拟 vs FluNet）

Step 5: PRCC 敏感性分析（1000次 LHS）
  └── 龙卷风图（Tornado plot）

Step 6: 三场景基准对比
  ├── 散发场景（outbreak threshold = 1例）
  ├── 聚集场景（outbreak threshold = 5例）
  └── 大流行场景（outbreak threshold = 20例）

Step 7: 干预方案优化
  ├── 网格扫描（78,125 种组合）
  └── 输出：热力图 + Pareto 前沿图
```
